## Resume Analysis using gemini api ##
The tech stack I have used in the project 

## For backend 

    FastAPI (Python)
    Python 3
    SQLAlchemy
    SQLite 
    CORS (FastAPI CORSMiddleware for frontend-backend communication)
    And for the file Handling, used local storage (static_uploads/ folder for saving uploaded resumes)

## For frontend  used react framework 

1. git clone https://github.com/poojithacodes/resume-llm.git 
    cd backend
    pip install -r requirements.txt 
    and run the backend

    cd frontend
    npm install
    npm run dev
